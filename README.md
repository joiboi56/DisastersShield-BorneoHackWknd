## General Description
### What the Project Does
DisasterShield is an intelligent disaster management mobile application designed to help communities stay informed and respond quickly during emergencies. The platform combines verified disaster updates, real-time hazard reporting, and emergency coordination tools in one system.

Key features include:
* Verified Government Updates – Provides trusted disaster information from official sources.
* Smart Risk Dashboard – Displays disaster risk levels based on the user’s current location.
* Rescue GPS Tracker – Allows users to share their real-time location to assist rescuers.
* Live Hazard Map – Enables community members to report hazards such as floods, blocked roads, or landslides in real time.
* AI Misinformation Filter – Detects potentially false or misleading disaster information by comparing user reports with verified sources and community data.

The goal of DisasterShield is to reduce confusion during disasters and support faster, safer decision‑making.

### Sustainable Development Goals (SDG) Addressed
This project supports the United Nations Sustainable Development Goals (SDGs):

* SDG 11 – Sustainable Cities and Communities
DisasterShield improves disaster preparedness and emergency response by providing reliable information and real-time hazard monitoring.

### Target Users
* Communities living in disaster‑prone areas
* Government disaster management agencies
* Emergency responders and rescue teams
* NGOs and humanitarian organizations

Since DisasterShield is currently presented as a prototype interface, no installation or software setup is required.

### Requirements
* Internet connection
* A modern web browser (Google Chrome, Microsoft Edge, Safari, or Firefox)

## How to Interact With the Prototype
### Step‑by‑Step Guide for Judges
1. Open the Application
   * Launch the DisasterShield prototype.
2. View the Smart Risk Dashboard
   * The system analyzes environmental data and hazard reports to show the disaster risk level based on your current location.
3. Check Verified Government Updates
   * Navigate to the news section to view trusted disaster updates from official sources.
4. Explore the Live Hazard Map
   * Users can view reported hazards such as floods or blocked roads on the map.
5. Submit a Hazard Report
   * Click the reporting button and submit a new hazard report to simulate community participation.
6. Test the Rescue GPS Tracker
   * Enable location sharing to simulate how victims can send their location to rescuers.
7. Try the AI Misinformation Filter
   * Enter a disaster-related message or report.
   * The system analyzes the information and checks it against verified sources and community reports.
